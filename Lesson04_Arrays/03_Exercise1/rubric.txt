Lab B Grading Rubric (15 marks)

You may receive zero if:
- Your code does not compile
- Your submission is in the wrong format
- You include any autogenerated package declarations, etc.
- You do not use the specified visibility, names, etc.
- You submit any files other than SimpleArrays.java

Each method is expected to meet the requirements provided in the instructions.
The exercise will be evaluated as follows:

Constructor Tests (2 marks):
1 mark: SimpleArrays constructor creates a non-null object from the default String [DefaultConstructorTest]
1 mark: SimpleArrays constructor creates a non-null object when given an input String [ArgumentConstructorTest]

arrayConcat Tests (5 marks):
1 mark: arrayConcat throws an IndexOutOfBoundsException when given an index that exceeds the array bounds [ConcatExceptionTest]
1 mark: arrayConcat returns the correct default array elements separated by # when a valid index is provided [DefaultConcatGivenIntegerTest]
1 mark: arrayConcat returns the correct input array elements separated by # when a valid index is provided [StringConcatGivenIntegerTest]
1 mark: arrayConcat returns the default String repeated four times with # between repetitions when no index is provided [DefaultConcatNoArgumentTest]
1 mark: arrayConcat returns the default input String repeated four times with # between repetitions when no index is provided [StringConcatNoArgumentTest]
